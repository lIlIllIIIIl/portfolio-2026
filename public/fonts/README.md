# Polices locales

Place tes fichiers de polices ici :

- **Geist (sans)** : `GeistVF.ttf` (variable font)
- **Geist Mono** : `GeistMonoVF.ttf` (variable font)
- **Noto Emoji** : `NotoColorEmoji.ttf` (classe `.emoji` dans `src/style.scss`)

Téléchargement : [Vercel Geist](https://vercel.com/font) ou [GitHub geist-font](https://github.com/vercel/geist-font). Noto Emoji : [GitHub noto-emoji](https://github.com/googlefonts/noto-emoji) (dossier `fonts/`).

Si tes fichiers ont d’autres noms, mets à jour les chemins dans `src/style.scss` (règles `@font-face`).
